A simple command-line application built in Python to help users track their pets, including feeding logs with timestamps.

- Add and view pets
- Log feedings
- Automatically timestamp feeding logs
- View feeding history
- Simple menu-based interface

## Why I Built This
I built this project to practice Python fundamentals, data structures, and real-world application logic while creating something practical and meaningful.

## How to Run

1. Clone the repo:
```bash
git clone https://github.com/vstjohn/pet_tracker.git
cd pet_tracker
```
2. Run the app:
```bash
python main.py
```
